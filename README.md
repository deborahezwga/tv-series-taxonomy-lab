# tv-series-taxonomy-lab
该仓库专门记录电视剧类站点在分类体系设计与页面层级方面的公开资料，通过文字方式归档剧集标签规则、年份分区逻辑以及关键词排布方式，形成结构研究档案。项目不涉及影视资源本身，仅整理公开页面信息元素。通过系统化整理，可用于分析不同站点在内容聚合与索引布局方面的差异。所有资料来源于互联网公开环境，仅供信息汇总与结构分析参考，不代表任何形式的推荐或合作关系说明。
